🔍 DIAGNÓSTICO: BOTÃO "Salvar e Avançar" NÃO FUNCIONA

O botão fica DESABILITADO quando qualquer uma dessas condições é verdadeira:
- disabled={isSaving || isLoadingData || pendingItemsToSaveCount === 0}

📋 POSSÍVEIS CAUSAS:

1️⃣ MAIS PROVÁVEL: pendingItemsToSaveCount === 0
   ❌ Você não adicionou nenhum item à lista
   ❌ Ou os itens têm quantidade = 0
   ❌ Ou os itens não têm status = 'Pendente'

2️⃣ MENOS PROVÁVEL: isLoadingData = true
   ❌ Sistema ainda carregando insumos/categorias
   ❌ Problemas com índices causando travamento

3️⃣ IMPROVÁVEL: isSaving = true
   ❌ Sistema travado em estado de salvamento

🔧 COMO VERIFICAR:

1. ABRA o console do navegador (F12 → Console)
2. COLE este código e pressione Enter:

```javascript
// Debug do botão Salvar e Avançar
console.log('🔍 DIAGNÓSTICO DO BOTÃO:');

// Verificar se o botão existe
const button = document.querySelector('button:contains("Salvar e Avançar")') || 
               Array.from(document.querySelectorAll('button')).find(btn => 
                 btn.textContent.includes('Salvar e Avançar') || 
                 btn.textContent.includes('Salvar e Cotar')
               );

if (button) {
  console.log('✅ Botão encontrado:', button.textContent);
  console.log('❌ Está desabilitado?', button.disabled);
  console.log('📋 Classes CSS:', button.className);
  
  // Verificar contador de itens
  const text = button.textContent;
  const match = text.match(/\((\d+)\)/);
  if (match) {
    const count = parseInt(match[1]);
    console.log('📊 Itens pendentes:', count);
    
    if (count === 0) {
      console.log('❌ PROBLEMA: Nenhum item na lista!');
      console.log('🔧 SOLUÇÃO: Adicione itens com quantidade > 0');
    } else {
      console.log('✅ Items na lista OK');
    }
  }
} else {
  console.log('❌ Botão não encontrado na página');
}

// Verificar se há itens na lista visualmente
const listItems = document.querySelectorAll('[class*="list"], [class*="item"]');
console.log('📋 Elementos de lista encontrados:', listItems.length);

// Verificar inputs de quantidade
const quantityInputs = document.querySelectorAll('input[type="number"], input[placeholder*="quantidade" i]');
console.log('🔢 Inputs de quantidade:', quantityInputs.length);

quantityInputs.forEach((input, i) => {
  console.log(`   Input ${i + 1}: valor = "${input.value}"`);
});
```

✅ Adicione itens à lista do lado esquerdo
✅ Certifique-se que a quantidade > 0  
✅ Verifique se os itens estão sendo salvos corretamente

SE isLoadingData = true:
✅ Aguarde o carregamento terminar
✅ Crie os índices faltantes (notifications principalmente)
✅ Recarregue a página

SE isSaving = true (travado):
